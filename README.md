#  Smart Agriculture Market Booking System

##  Overview

The Smart Agriculture Market Booking System is a web application designed to help farmers book time slots in agricultural markets (mandis). It reduces waiting time, manages vehicle queues, and improves the selling process for farmers.

---

##  Features

*  Slot booking for selling crops
*  Vehicle entry and queue management
*  Market availability tracking
*  Crop price display (basic)
*  Farmer and market management

---

##  Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Node.js, Express
* **Database:** MongoDB

---

##  How to Run

### 1. Install Dependencies

```bash
npm install
```

### 2. Start MongoDB

```bash
mongod
```

### 3. Run Backend

```bash
node server/server.js
```

### 4. Run Frontend

Open:

```
client/index.html
```

---

##  API Endpoints

* `/api/farmers` → Get farmers
* `/api/markets` → Get markets
* `/api/bookings` → Get bookings

---

##  Objective

To reduce congestion in markets and help farmers sell crops efficiently using digital booking.
